Controler les informations affichées à votre sujet dans les services Cinéwaves

Vous pouvez rendre certaines informations de votre compte Cinéwaves privées ou visibles par tous. Ainsi, vous contrôlez qui voit les informations vous concernant (comme votre date de naissance ou votre numéro de téléphone) dans les services Cinéwaves.

Définir quelles informations afficher

Les autres utilisateurs des services Cinéwaves peuvent voir votre nom et votre photo de profil lorsque vous utilisez votre profil de compte, y compris lorsque vous communiquez ou partagez du contenu.

Cliquez sur Informations personnelles.

Sous "Choisissez ce que les autres voient", cliquez sur Accéder à la page "À propos de moi".
En dessous d'un type donné d'informations, vous pouvez définir qui peut actuellement les voir.
Sélectionnez l'une des options suivantes :
Pour rendre ces informations privées, cliquez sur "Vous uniquement" Privé, appuyer pour modifier l.
Pour rendre ces informations visibles par tous, cliquez sur "Tout le monde" Personnes.

Ajouter, modifier ou supprimer des informations personnelles

Important : Certaines informations ne peuvent pas être supprimées de votre compte Cinéwaves. Par exemple, vous pouvez modifier votre nom et votre date de naissance, mais pas les supprimer.

1. Cliquez sur Informations personnelles.
2. Sous "Choisissez ce que les autres voient", cliquez sur Accéder à la page "À propos de moi".
3. Modifiez les informations vous concernant :
Ajouter : pour chaque catégorie à laquelle vous voulez ajouter des informations, cliquez sur Ajouter un utilisateur Ajouter.
Modifier : cliquez sur les informations que vous voulez modifier, puis sur Modifier Modifier.
Conseil : Si vous avez changé votre nom récemment, vous devrez peut-être attendre avant de pouvoir le modifier de nouveau.
Supprimer : cliquez sur les informations que vous voulez supprimer, puis sur Supprimer Supprimer.
4. Suivez les instructions à l'écran.

Afficher et gérer vos profils dans les services Cinéwaves

Dans certains services Cinéwaves, les autres utilisateurs peuvent voir votre profil. Vous pouvez trouver les profils de certains services dans votre compte Cinéwaves.

1. Cliquez sur Informations personnelles.
2. Faites défiler la page jusqu'à "Vos profils". Cliquez ensuite sur Voir les profils.
3. Sélectionnez un service pour afficher les informations de votre profil.
4. Accédez au service pour gérer les informations de votre profil.

Découverte du profil

Lorsque des utilisateurs vous recherchent à l'aide de votre numéro de téléphone ou de votre adresse e-mail dans des services Cinéwaves, la découverte du profil permet de déterminer ce qu'ils verront. Lors de la configuration, vous indiquez si les personnes avec lesquelles vous n'avez pas interagi sur les services Cinéwaves, mais qui disposent de vos coordonnées, peuvent voir votre photo de profil et votre nom complet ou abrégé. Lorsque vous interagissez avec une personne (par exemple, lorsque vous communiquez dans un forum ou partagez un album dans commentaire), elle voit généralement votre nom complet et votre photo de profil issus de votre compte Cinéwaves.

Avec la découverte du profil, vous pouvez :
. Utiliser votre nom tel qu'il figure dans votre profil principal ou une version abrégée ;
. Afficher ou masquer votre photo de profil.

Gérer la découverte du profil

Important : Pour permettre aux utilisateurs de vous trouver grâce à votre numéro de téléphone, activez l'option Permettre de vous trouver avec votre n° de téléphone dans les paramètres de votre numéro de téléphone.

1. Accédez à votre compte Cinéwaves.
2. Cliquez sur Informations personnelles.
3. Sous "Vos profils", cliquez sur Voir les profils.
4. Sous votre photo et votre nom, cliquez sur Paramètres de découverte du profil.
5. Activez l'option Découverte du profil.
6. Pour modifier l'affichage de votre nom et de votre photo sur votre profil, cliquez sur Modifier.
